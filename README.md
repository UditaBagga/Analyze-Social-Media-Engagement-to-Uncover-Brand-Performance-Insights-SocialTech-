# Analyze-Social-Media-Engagement-to-Uncover-Brand-Performance-Insights-SocialTech-
Data analysis project using Excel, SQL, Python, Machine Learning and Power BI to analyze social media engagement patterns and generate business insights.

# Social Media Engagement Analysis

## Objective
To analyze social media engagement patterns and identify factors influencing high-performing content.

## Tools Used
Python
SQL
Excel
Power BI
Machine Learning (Clustering)

## Project Workflow
1. Data Cleaning and Preparation
2. Exploratory Data Analysis using Python
3. SQL queries for engagement insights
4. Power BI dashboard creation
5. Machine learning clustering to identify engagement segments

## Key Insights
• Posting time significantly affects engagement levels  
• Certain content categories receive higher interaction  
• Sponsored posts show different engagement behavior  

## Business Impact
These insights help optimize social media strategy and improve audience engagement.

## Files in the Repository
Dataset – raw data used for analysis  
Python – data cleaning and EDA  
SQL – engagement analysis queries  
Power BI – dashboard for visualization  
Machine Learning – clustering model
